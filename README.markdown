# quote_unquote
Wraps strings with quotes.

## Installation

```ruby
gem install quote_unquote
```

## Examples

```ruby
require 'quote_unquote'

"hello".q     # 'hello'
"hello".qq    # "hello"

"'hello'".uq  # hello
'"hello"'.uqq # hello
```
